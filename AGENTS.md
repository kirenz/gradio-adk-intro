# Instructions

This is a tutorial on how to use Google Agents Development Kit (ADK) with Gradio as an agent interface.

The project can be run with `uv run adk web` as well as Gradio interface with `uv run app.py`. Both should be run from the project root directory.

## Python management with uv

The project is managed with uv. 

To add packages, use the command:

```
uv add <package-name>
```

To run the project, use the command:

```
uv run
```
